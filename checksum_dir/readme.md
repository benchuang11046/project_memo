# Program Usage

## python version
python-3.6 later


## install requirements package
```
pip install -r requirements.txt
```

## Execute checksum_dir
```
python checksum_dir.py [A_dir] [B_dir]
```

### The same result
```
md5 same
sha1hash same
```

### Not the same result
```
md5 not same
sha1hash not same
```
